# Repositório de Códigos de Arduino e VHDL para Sistemas Digitais
Este é um repositório dedicado aos códigos de Arduino e VHDL desenvolvidos durante a matéria de Sistemas Digitais do curso de Engenharia de Computação. Aqui você encontrará códigos que abrangem desde os conceitos básicos até projetos mais avançados, tudo em um só lugar.

## Sobre a matéria
A disciplina de Sistemas Digitais é uma das mais importantes na área de Engenharia de Computação. Ela aborda conceitos fundamentais de eletrônica digital, como lógica combinacional, lógica sequencial, registradores, contadores, circuitos integrados programáveis e microcontroladores. Com o conhecimento adquirido nessa matéria, é possível desenvolver soluções digitais para uma ampla variedade de aplicações.

## Sobre o repositório
Este repositório contém códigos de Arduino e VHDL que podem ser utilizados como referência e/ou base para seus próprios projetos. Os códigos estão organizados em pastas por temas e conceitos abordados na matéria, para facilitar a navegação e a busca. Todos os códigos foram testados e estão funcionais.

## Como usar o repositório
Para utilizar os códigos deste repositório, basta clonar o repositório em sua máquina e abrir os arquivos com a IDE de programação adequada. Caso tenha dúvidas ou precise de ajuda, sinta-se à vontade para abrir uma issue ou entrar em contato com o responsável pelo repositório.
